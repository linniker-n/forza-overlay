# Forza Overlay

Baixe e execute:

```text
Forza Overlay 1.0.0.exe
```

## Configuracao do Forza Motorsport

```text
Configuracoes -> HUD e Gameplay -> Data Out (UDP)
  Habilitar : SIM
  IP        : 127.0.0.1
  Porta     : 5300
  Formato   : Dash
```

Use o jogo em modo **Janela Sem Bordas**.

Ao abrir o app, clique em **Personalizar layout** se quiser ajustar o HUD antes de jogar. O overlay aparece automaticamente quando a corrida comecar.
